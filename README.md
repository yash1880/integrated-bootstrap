# 💇‍♀️ LADIVA Salon Website

A modern and fully responsive **Salon & Beauty Website** developed using **React.js** and **Vite**.  
This project is inspired by a professional **ThemeForest salon template** and designed for beauty salons, spas, skincare, nail studios, and cosmetic businesses.

The website contains multiple pages, reusable components, and organized assets for a clean and scalable frontend structure.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ladiva-salon.git
cd ladiva-salon
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

# ▶️ Run Project

## 3️⃣ Start Development Server

```bash
npm run dev
```
